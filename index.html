Hattrick-enhanced
by rainworm

Greasemonkey scripts for Hattrick that takes Hattrick to another level.

So far script can
- Display team players avatars even if you don't support the game developers,
- hide annoying ads on page.

Greasemonkey scripts for Hattrick are developed here. I wan't to add some game functionality and improve game experience.

Install

It is asumed that you already have Firefox browser on your computer. To use this scripts you have to install Greasemonkey (add-on for Firefox).

How scripts works

Basically, when you go to a web site, Greasemonkey kicks into action and loads the scripts as the page loads, changing how the HTML code of the site interacts with the browser to display the changed web site for you.
